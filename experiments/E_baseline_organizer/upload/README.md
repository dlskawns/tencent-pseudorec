- Expected wall (suggested): ~TBD min

## 2. What this code does
Trains the organizer's PCVRHyFormer (multi-domain sequence encoders →
per-domain query decoders → joint token fusion via RankMixer) with two
patches:

- **`label_time`-aware split** (instead of organizer's row-group order)
  — prevents exposure → feedback leakage and works on single-row-group
  parquet files.
- **10% user_id OOF holdout** (seed=42) — held out from BOTH train and
  valid so the reported `best_oof_AUC` is a true generalization estimate.

Schema is auto-generated by `make_schema.py` if missing — vocab sizes,
list dims, and per-domain `ts_fid` are inferred from the parquet metadata.

## 4. Platform contract
The platform must set:
- `TRAIN_DATA_PATH` — directory containing training `*.parquet`.
- `TRAIN_CKPT_PATH` — writable directory for checkpoints + `metrics.json`.
- (optional) `TRAIN_LOG_PATH`, `TRAIN_TF_EVENTS_PATH` — derive from CKPT if unset.

Then invoke `bash run.sh`.

## 5. Outputs
- `${TRAIN_CKPT_PATH}/metrics.json` — primary reporting source. Includes
  `best_val_AUC`, `best_oof_AUC`, `split_meta`, `seed`, `git_sha`,
  `config_sha256`, `host`.
- `${TRAIN_CKPT_PATH}/global_step*.best_model/` — model.pt + schema.json
  + train_config.json (used by inference).
- `${TRAIN_LOG_PATH}/train.log` — full training log.

## 6. Sanity dry-run
For a 30-second pipeline check before the full run:
