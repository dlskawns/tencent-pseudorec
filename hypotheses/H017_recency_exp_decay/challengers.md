# H017 — Challenger Frames

> H015 sub-form variant. primary_category = `temporal_cohort` (H015 와 같음).

## Frame A (default)
exp curve 가 linear 보다 recent emphasis 강함 → cohort drift 처리 더 효과적
가능. Δ vs H015 측정으로 form 효과 분리.

## Frame B (counter)
form 변경 효과 거의 없음 (mean weight 동일 1.0, 분포만 약간 다름). linear 가
이미 충분 또는 cohort drift 처리 mechanism class 자체 marginal.

## Frame C (orthogonal)
form 보다 range 가 더 중요. [0.5, 1.5] 너무 보수적, [0.2, 5.0] 같은 wider
range 가 진짜 효과. — H018+ sub-H.

## Decision

A 선택 — H015 paired 비교로 form 효과 단독 측정. cost 0 (H015 동시 launch).
